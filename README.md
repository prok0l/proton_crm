### Cтарт проекта:
- создание .env файла
    ```dotenv
    SECRET_KEY=
    TOKEN=
    ADMIN_IDS=
    USE_REDIS=False
    REMIND_TIME=60
    BOT_LINK=
    ```
- запуск проекта
    ```shell
    docker-compose up
    ```
